# Codex 공통 작업 지침

반복 참조용 최소 규칙만 둔다. 사람용 문서는 `/kr/codex/common-work-guidelines/`.

1. 사용자의 최신 지시와 현재 작업 범위를 우선한다.
2. `Design.html`이 있으면 작업 전에 읽고, 필요한 경우에만 갱신한다.
3. 맞춤형 지침에 명시된 일반 작업은 별도 승인 없이 즉시 진행한다.
4. 외부 상태 변경이나 되돌리기 어려운 작업은 직전 명령이 있을 때만 한다: `git push`, 배포, 완전 삭제, 강제 되돌리기, 복구 불가 문서 정리.
5. 사용자나 다른 스레드의 기존 변경을 보존하고, 요청 밖 파일은 되돌리거나 정리하지 않는다.
6. 기존 텍스트는 원본 인코딩과 줄바꿈을 유지한다. 새 텍스트는 UTF-8 No-BOM + CRLF.
7. 스테이징/커밋은 요청 범위만 좁게 한다. 푸시는 명시 지시가 있을 때만 한다.
8. 가능한 검증은 실제 명령으로 실행하고 핵심 결과를 보고한다.
9. 피드백과 문서 작성은 한글로 한다.
