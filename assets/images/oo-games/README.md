# OO Games CI assets

The controller-shaped `oo` symbol and the `games` wordmark are separate assets so Unity can animate scale and opacity independently.

- `oo-games-symbol.svg`: editable symbol master
- `oo-games-wordmark.svg`: editable wordmark master
- `oo-games-lockup.svg`: editable combined master
- `oo-games-symbol-4096.png`: transparent Unity symbol sprite
- `oo-games-wordmark-4096.png`: transparent Unity wordmark sprite
- `oo-games-lockup-4096.png`: transparent static web/print lockup
- `oo-games-lockup-4096.webp`: compressed static web lockup
- `*-white-4096.png`: reversed variants for dark backgrounds

The SVG wordmark specifies Arial Rounded MT Bold with Arial and generic sans-serif fallbacks. The exported PNG and WebP files preserve the approved Windows rendering independently of installed web fonts.
